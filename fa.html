ایده این صفحه وب نشان دادن زیبایی طراحی با حالت روشن و
حالت تاریک است. برای تغییر زبان به انگلیسی کلیک کنید.

نمونه‌هایی از دوره‌های ما:

دوره‌های HTML و CSS

دوره‌های "HTML و CSS" معمولاً اصول توسعه وب را پوشش می‌دهند. این دوره‌ها به شما یاد می‌دهند چگونه صفحات وب را با استفاده از HTML (زبان نشانه‌گذاری فرامتن) و CSS (شیوه‌نامه‌های آبشاری) ایجاد و سبک‌دهی کنید. همچنین، نحوه ساختاردهی محتوا، اعمال استایل‌ها و ایجاد صفحات وب واکنش‌گرا برای دستگاه‌های مختلف را فرا می‌گیرید. دانش‌آموزان می‌توانند طراحی وب‌سایت‌های جذاب را یاد بگیرند و مهارت‌های اساسی برای توسعه وب فرانت‌اند را کسب کنند.

دوره‌های C#

دوره‌های C# اصول برنامه‌نویسی با این زبان چندمنظوره را آموزش می‌دهند. C# برای توسعه وب‌سایت‌ها، برنامه‌ها، بازی‌های ویدئویی و بسیاری دیگر استفاده می‌شود. این دوره‌ها شامل مباحثی همچون نحو C#، انواع داده‌ها، ساختارهای کنترلی، برنامه‌نویسی شیءگرا و کار با چارچوب .NET می‌شوند. دانش‌آموزان می‌توانند نوشتن کد، ایجاد برنامه‌ها و کسب مهارت‌های اساسی برای توسعه نرم‌افزار را بیاموزند.

دوره‌های Adobe Photoshop

دوره‌های "Adobe Photoshop" برای آموزش استفاده از این نرم‌افزار قدرتمند ویرایش تصویر و طراحی گرافیک طراحی شده‌اند. این دوره‌ها شامل مباحثی مانند رتوش عکس، دستکاری تصویر، ایجاد آثار هنری دیجیتال، کار با لایه‌ها، ماسک‌ها، فیلترها و ابزارهای مختلف فتوشاپ می‌شوند. دانش‌آموزان می‌توانند بهبود عکس‌ها، ایجاد گرافیک‌های جذاب و مهارت‌های لازم برای طراحی و ویرایش دیجیتال را یاد بگیرند.
